## 下载

获取主题的方法：前往 [Releases](https://github.com/bhaoo/Cuckoo/releases) 或者 下方QQ群 下载主题

## 特点

- 全站PJAX+评论AJAX
- 响应式 / 自适应 
- 图片懒加载 Lazyload
- 文章目录树
- 用户UA以及仿Biibili认证
- 支持评论表情
- 代码高亮
- 支持多种联系方式
- 基于 Links 的友链页面
- B站追番独立页面
- 自定义静态文件源 / 随机图源 
- 自定义抽屉导航栏
